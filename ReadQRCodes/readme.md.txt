# Read QR Codes - Android Application

## Overview
**Read QR Codes** is a modern, fast, and lightweight Android application specifically designed to scan and decode QR codes with ease. The app is built with a focus on a clean User Experience (UX), high performance, and a comprehensive feature set for daily needs.

## Key Features
- **Real-time Scanning**: Utilizes the camera sensor to instantly detect and decode QR codes with high accuracy.
- **Custom Scanner UI**: Features a professional overlay design, scan line animations, and intuitive controls.
- **Import from Gallery**: Allows users to scan QR codes from images already stored on the device.
- **Flashlight Control**: A feature to toggle the camera's flash for scanning in low-light environments.
- **Multi-Language Support (Localization)**: Supports 9 international languages:
    - English (United States)
    - Indonesian
    - Chinese (Mandarin)
    - Japanese
    - Korean
    - Arabic
    - Spanish
    - French
    - German
- **Scan Result Management**:
    - **Copy**: Copy the scanned text to the clipboard.
    - **Share**: Share scan results with other apps (WhatsApp, Email, etc.).
    - **Open Link**: Automatically detects URLs and provides an option to open them directly in a web browser.
- **Material 3 Design**: Follows Google's latest design standards for an aesthetic and responsive interface.
- **Splash Screen**: An elegant app identity display shown during the initial app load.

## Technologies & Libraries Used
The application was developed using the **Java** programming language with several industry-standard third-party libraries:
- **ZXing Android Embedded**: The core engine for barcode and QR code processing and decoding.
- **Google Material Components**: For UI elements like CardView, BottomNavigation, and Buttons with Material 3 styling.
- **AndroidX**: Modern Android base components for stable compatibility.
- **Core-SplashScreen**: Implementation of Google's latest Splash Screen API.
- **Hanken Grotesk Font**: Custom typography to provide a modern and premium feel.
