# Text To Speech App

A simple and intuitive Android application that converts written text into spoken words using the Android Text-to-Speech (TTS) engine.

## Features

- **Text-to-Speech Conversion:** Easily convert any typed text into audible speech.
- **Multi-language Support:** Primarily supports Indonesian, with an automatic fallback to English (US) if Indonesian data is unavailable.
- **Theme Toggle:** Switch between Light and Dark modes for a comfortable viewing experience.
- **Splash Screen:** A smooth entry into the application.
- **User-Friendly Interface:** Built with Material Design components for a modern look and feel.
- **About Dialog:** Quick access to information about the application.

## How to Use

1. Open the app and wait for the splash screen to finish.
2. Enter the text you want to hear in the input field.
3. Tap the **"SPEAK"** button.
4. The app will speak the text and display it on the screen.
5. Use the menu in the toolbar to change the theme or view the "About" section.

## Technologies Used

- **Language:** Java
- **UI Framework:** Android XML with Material Components
- **API:** Android TextToSpeech (TTS)
- **Minimum SDK:** Depends on project configuration (usually API 24 or higher)

## Installation

1. Clone or download the project.
2. Open it in Android Studio.
3. Build and run the project on an emulator or a physical Android device.
