# Smart To Do List

Smart To Do List is a modern productivity application designed to help users organize their daily activities, monitor task deadlines, and increase efficiency in completing everyday work. With a clean and modern interface, this app helps users stay focused, organized, and productive at all times.

## Features

- **Task Management**: Easily create, update, and delete tasks.
- **Priority Levels**: Categorize tasks by priority (Low, Medium, High) with distinct color coding.
- **Deadline Tracking**: Set and monitor deadlines for every task to ensure timely completion.
- **Statistics Dashboard**: Gain insights into your productivity with a summary of total, completed, and pending tasks.
- **Search Functionality**: Quickly find specific tasks using the built-in search bar.
- **Modern UI/UX**: Built with Material Design 3 for a sleek, responsive, and user-friendly experience.
- **Theme Support**: Choose between Light Mode, Dark Mode, or follow the System theme.

## Getting Started

### Prerequisites
- Android Studio
- Android SDK 34 or higher

### Installation
1. Clone or download the source code.
2. Open the project in Android Studio.
3. Sync project with Gradle files.
4. Build and run the application on your device or emulator.

## Technologies Used
- **Language**: Java
- **UI Framework**: Material Components for Android (Material 3)
- **Database**: SQLite (Local storage for tasks)
- **Layouts**: ConstraintLayout, CoordinatorLayout
