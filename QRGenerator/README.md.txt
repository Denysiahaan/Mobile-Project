# QR Generator Application

## About the Application
QR Generator is a modern and intuitive Android application designed to help users create custom QR codes instantly. Whether you need to share a website URL, a piece of text, an email address, or a phone number, this app provides a seamless experience to generate and manage your QR codes.

## 📥 Download & Install
You can download the latest installer from the [Releases](https://github.com/Denysiahaan/Mobile-Project/releases) section of this repository.
1. Download the provided `.apk` file.
2. Transfer it to your Android device (if downloaded on PC).
3. Open the file and install (ensure "Install from Unknown Sources" is enabled in your device settings).

## Key Features
- **Multi-Type Support**: Generate QR Codes for URLs, plain Text, Emails, and Phone Numbers.
- **Instant Generation**: Create QR codes in real-time as you type or input data.
- **QR Preview**: View a live preview of the generated QR code before saving or sharing.
- **Save to Gallery**: Save your generated QR codes directly to your device's gallery for later use.
- **Easy Sharing**: Share your QR codes with others through various apps like WhatsApp, Telegram, or Email.
- **History Management**: Automatically save your generated QR codes in a local database to access them later.
- **Multilingual Support**: Supports multiple languages including English, Indonesian, Arabic, Chinese, Japanese, Korean, French, German, and Spanish.
- **Modern UI**: A clean, responsive, and user-friendly interface built with Material Design 3.
- **Animations**: Smooth transitions and animations for a premium user experience.

## Libraries and Technologies Used
- **Language**: Java
- **UI Framework**: XML with Material Design Components
- **ZXing (Zebra Crossing)**: The core library used for generating and decoding QR codes.
- **SQLite**: Used for local data storage to keep a history of saved QR codes.
- **View Binding**: For efficient interaction with UI components.
- **Android Jetpack**: Includes Components like Fragments, ViewModel, and Activity Results API.

## How to Use
1. Open the app and select the type of QR code you want to generate from the top chips (URL, Text, Email, or Phone).
2. Enter the required information in the input field.
3. Tap on **"Generate QR Code"**.
4. Once generated, you can choose to **Save** it to your gallery, **Share** it with others, or create a **New** one.
5. Access your previously generated codes in the **"Saved"** tab.
